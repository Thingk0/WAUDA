# WAUDA: 혼자 뭐하니?


## Introduction

학교 학생들이 급하게 필요한 다양한 모임을  모집할 때 사용하기 적합한 플랫폼 서비스입니다. 이 웹 애플리케이션은 모임 구성이 필요한 상황에 최적화되어 있으며, 예를 들어 공부팟, 택시팟, 점심팟 등 다양한 파티를 손쉽게 만들 수 있습니다.


## Preparation

본 프로젝트는 스프링 프레임워크를 기반으로 제작된 애플리케이션입니다.

* Java 17
* JDK corretto 17
* SpringBoot 2.7.12 ver.
* JPA 'Hibernate'
* Thymeleaf
* H2
* MySQL
* BootStrap 5.1.3 ver.
* JQuery 3.7.0 ver.
